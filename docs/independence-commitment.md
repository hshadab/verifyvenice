# Independence Commitment

## Statement

This verification was conducted independently. The findings are
published regardless of whether they are favorable or unfavorable
to Venice.ai.

## Commitments

1. **No editorial control:** Venice.ai had no input into the
   methodology, analysis, prompts, thresholds, or conclusions.

2. **Full publication:** All results are published, including any
   negative findings (evidence of filtering, model substitution,
   or hidden prompt injection).

3. **Open methodology:** The complete source code, configuration,
   and analysis pipeline are publicly available for independent
   verification and reproduction.

4. **Raw data availability:** Anonymized raw API response data
   (logprobs, token distributions, timing) is published alongside
   the report. (Response text may be summarized rather than
   reproduced in full.)

5. **Reproducibility:** Any researcher with API access to Venice.ai
   and Together.ai can reproduce our tests using the published
   configuration and scripts.
